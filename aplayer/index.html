Место для содержимого.

Description:

This scenario will play a Media audio stream. This category is typically used to play any audio file, apart from movies. Select an audio file from your library (e.g. a song), and press play. It will have similar characteristics to the Movie category, but the audio effects that are applied to each audio category might be different.

To observe how each stream category interacts with the others, run both the Audio Category Sample and the Audio Category Companion sample side by side.
